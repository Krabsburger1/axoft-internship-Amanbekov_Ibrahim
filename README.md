Запуск проекта
Задания A1–A4 — это статические HTML/CSS/JS файлы, фреймворки не используются.
Способ 1 — напрямую через браузер:
Открыть файл index.html из нужной папки в браузере (двойной клик или перетащить в окно браузера).
Способ 2 — через Live Server (VS Code):

Установить расширение Live Server в VS Code
Открыть папку задания в VS Code
Нажать правой кнопкой на index.html → Open with Live Server

Способ 3 — через локальный сервер (Node.js):
bashnpx serve .
Затем открыть http://localhost:3000 в браузере.

Рефлексия
Что было сложно:
Сложнее всего далась реализация floating label эффекта в задании A4 — нужно было точно совместить CSS transform с transition, чтобы анимация выглядела плавно. Также потребовалось время на настройку адаптивной сетки в A3 под все три брейкпоинта.
Что понравилось:
Понравилось работать с CSS Grid и Flexbox — видно, как из простых свойств получается полноценный адаптивный интерфейс. Приятно было реализовать анимации через @keyframes без сторонних библиотек.
